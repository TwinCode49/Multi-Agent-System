# Agent Performance Metrics

Evaluates agent and skill quality across three dimensions:

- **Structural**: frontmatter validity, section completeness, keyword count, Do NOT rules
- **Functional**: integrates with agent-testing framework for test pass rate
- **System**: cross-platform sync between `{{agentsDir}}` and `{{githubSkillsDir}}`

## Usage

```bash
node tools/agent-metrics/report.mjs
node tools/agent-metrics/report.mjs --save   # Save report to reports/ directory
```
