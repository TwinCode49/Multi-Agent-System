# Multi-Agent Workflows

Defines, validates, and executes multi-agent workflows with dependency resolution, parallel execution, and orchestrator synthesis.

## Validating & Running

```bash
node tools/agent-workflows/run.mjs                    # Validate all workflow definitions
node tools/agent-workflows/executor.mjs --submit workflow-name   # Submit for execution
node tools/agent-workflows/executor.mjs --status <run-id>       # Check status
node tools/agent-workflows/executor.mjs --list                  # List all runs
node tools/agent-workflows/executor.mjs --handoff <run-id>      # View handoff chain
```

## Workflow Definitions

Each `.json` file in `definitions/` defines a workflow with:
- `name`, `version`, `description`
- `trigger_keywords` for dispatch matching
- `steps` array with `id`, `agent`, `prompt`, `depends_on`
- Optional `synthesizer` for orchestrator synthesis
