# Agent Testing Framework

Validates agent and skill definitions for structural correctness, keyword coverage, section completeness, and cross-platform sync.

## Usage

```bash
node tools/agent-testing/run.mjs
```

## What It Checks

- **Frontmatter**: required fields present (description, mode, permission, model for read-only)
- **Keywords**: TRIGGER KEYWORDS count ≥ 5
- **Sections**: Core Responsibilities, Constraints, Behavior Rules, Handoff Protocol
- **References**: `references/` directory has `.md` files for each skill
- **Cross-platform sync**: skills synced between `.opencode/agents` and `.github/skills`
