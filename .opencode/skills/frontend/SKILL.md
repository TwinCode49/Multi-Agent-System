# Frontend Skill

## Goal
Build accessible, responsive, and performant user interfaces with clean component architecture.

## Design Principles
- Mobile-first responsive design
- Atomic design methodology (atoms → molecules → organisms)
- Accessible by default (WCAG 2.1 AA minimum)
- Design tokens for consistent theming

## Component Architecture
- Single responsibility per component
- Props interface defined before implementation
- Consider all states: loading, empty, error, edge cases
- Separate logic from presentation where possible

## Constraints
- Do NOT use pixel values where relative units (rem, em, %) are appropriate
- Do NOT skip keyboard navigation support
- Do NOT use generic AI aesthetics
- Do NOT add animations without performance consideration
