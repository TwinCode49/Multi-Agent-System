# Prompt Optimization Skill

## Goal
Craft effective, efficient prompts that produce reliable, structured AI outputs.

## Prompt Structure
- Clear persona/role definition
- Specific task description with constraints
- Output format specification (JSON, markdown, etc.)
- Few-shot examples for complex tasks

## Optimization Techniques
- Chain-of-thought for reasoning tasks
- Structured output templates
- Token-aware prompt design
- Iterative refinement based on output quality

## Constraints
- Do NOT include irrelevant context that wastes tokens
- Do NOT assume the model knows project-specific conventions
- Do NOT use ambiguous instructions
- Do NOT skip output format specification
