# Agent Catalogs & Sources

Unlike skills, there is no centralized catalog like skills.sh for agents. Sources are dispersed across repositories and frameworks.

## Agent Definitions & Examples

| Source | Description |
|---|---|
| [microsoft/skills](https://github.com/microsoft/skills) | Includes `.github/agents/*.agent.md` examples |
| [aiagentskills/skills](https://github.com/aiagentskills/skills) | Includes subagent definitions in `subagents/` |
| [obra/superpowers](https://github.com/obra/superpowers) | Agent workflow framework with subagent patterns |
| [hugojosefson/opencode-config](https://github.com/hugojosefson/opencode-config) | Example agent definitions for OpenCode |
| [addyosmani/agent-skills](https://github.com/addyosmani/agent-skills) | Agent setups with AGENTS.md orchestration |

## Orchestration Frameworks

| Framework | Description |
|---|---|
| [oh-my-openagent (OMO)](https://github.com/opensoft/oh-my-openagent) | Multi-agent orchestration with ultrawork mode, parallel execution, keyword detection |
| [opencode-council](https://github.com/marcel-tuinstra/opencode-council) | Multi-agent deliberation with CTO/DEV/PM roles |
| [Pocket-Universe](https://github.com/unknown/pocket-universe) | Subagent messaging and fire-and-forget patterns |

## Agent Directories by Platform

| Platform | Default Agent Directory |
|---|---|
| OpenCode | `.opencode/agents/` or `~/.config/opencode/agents/` |
| Antigravity | `~/.gemini/antigravity/agents/` |
| VS Code | `.github/agents/` |
